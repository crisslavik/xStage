# Converter dependencies (optional, for format support):
# - FBX: Requires USD FBX plugin (usdFbx) or Adobe USD Fileformat Plugins

#   * Installation: Build from source or install pre-built binaries
#   * xStage automatically detects and uses Adobe plugins if installed
# - Alembic: Requires USD Alembic plugin (usdAbc, usually included with USD) or Python Alembic library

#   * Must be compiled and installed separately (not in requirements.txt)
