# Code of Conduct

## Our Pledge

We pledge to make participation in xStage a harassment-free experience for everyone.

## Our Standards

- Be respectful and inclusive
- Accept constructive criticism gracefully
- Focus on what's best for the community
- Show empathy

## Enforcement

Report violations to: conduct@xstage-pipeline.org

This Code of Conduct is adapted from the Contributor Covenant v2.1.
